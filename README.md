# Pizza Point

Pizza Point is a full-stack food ordering web application built using the **MERN Stack (MongoDB, Express.js, React, Node.js)**. The platform allows users to browse menu items, add food to the cart, and place orders easily with secure payment integration.

---

## Features

- User Authentication (Signup / Login)
- Browse Food Menu
- Add to Cart & Order Food
- Razorpay Payment Integration
- Cash on Delivery Option
- Responsive Design (Mobile + Desktop)
- Admin Panel for Menu and Order Management

---

## Tech Stack

**Frontend**
- React.js
- HTML
- CSS
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Payment Gateway**
- Razorpay
